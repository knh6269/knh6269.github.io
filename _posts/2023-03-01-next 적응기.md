# 프레임워크와 라이브러리의 차이점
> 라이브러리 : 사용자가 파일 이름이나 구조 등을 정하고, 모든 결정을 내림 
 
> 프레임워크 : 파일 이름이나 구조등을 정해진 규칙에 따라 만들고 따름 

```
내가 만들지 않은 코드를 "가지고 와서" 내가 무언가 Customize를 하고 있다면 👉 LIbrary.

내가 약속된 특정 영역에 코드를 짜니까, 그 코드를 "가지고 가서" 알아서 뭔가 뚝딱 만들어준다면? 👉 Framework.
```
